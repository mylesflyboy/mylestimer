TimerPlugin

TimerPlugin is a lightweight Minecraft timer plugin designed for challenge servers (similar to BastiGHG-style challenges).
It displays a global timer for all players above the XP bar in the ActionBar and supports both count-up and count-down modes.

The timer is bold, minimalistic, and distraction-free, showing only the formatted time (HH:MM:SS).

Built for Java 21 and compatible with Purpur 1.21.11, while also working on most Paper / Spigot based servers.

Features

• Global timer visible to all players
• Displays time in HH:MM:SS format
• Bold ActionBar display for clear visibility
• Count-up mode (challenge timer)
• Count-down mode (event timer)
• Pause and resume the timer anytime
• Reset the timer instantly
• Command tab completion support
• Lightweight and optimized
• No configuration required

Commands
Command	Description
/timer up	Starts a count-up timer from 00:00:00
/timer down <seconds>	Starts a count-down timer
/timer stop	Pauses the timer
/timer resume	Resumes the paused timer
/timer reset	Stops and resets the timer
/timer reload	Reloads the timer state
